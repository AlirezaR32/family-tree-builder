# 🚀 راهنمای سریع شروع

## نصب و راه‌اندازی

### روش اول: نصب خودکار

```bash
./install.sh
```

### روش دوم: نصب دستی

#### بک‌اند
```bash
cd backend
pip install -r requirements.txt
python app.py
```

#### فرانت‌اند (در ترمینال جدید)
```bash
cd frontend
npm install
npm run dev
```

## اجرای پروژه

### روش سریع
```bash
./run.sh
```

### روش دستی
```bash
# ترمینال 1 - بک‌اند
cd backend
python app.py

# ترمینال 2 - فرانت‌اند
cd frontend
npm run dev
```

## آدرس‌ها

- 🌐 فرانت‌اند: http://localhost:3000
- 🔌 API بک‌اند: http://localhost:5000

## تست کردن

```bash
python test_family_tree.py
```

## اولین قدم‌ها

1. مرورگر را باز کنید و به `http://localhost:3000` بروید
2. روی دکمه "📥 بارگذاری داده نمونه" کلیک کنید
3. درخت خانوادگی نمونه را مشاهده کنید
4. در بخش "🔍 یافتن نسبت خانوادگی" دو فرد را انتخاب کنید
5. الگوریتم "مقایسه BFS و DFS" را انتخاب کنید
6. روی "🔍 جستجوی نسبت" کلیک کنید

## مثال عملی

### افزودن خانواده جدید

1. **افزودن پدربزرگ**
   - شناسه: `grandfather`
   - نام: `حسین`
   - جنسیت: `مرد`

2. **افزودن مادربزرگ**
   - شناسه: `grandmother`
   - نام: `فاطمه`
   - جنسیت: `زن`

3. **افزودن رابطه همسری**
   - همسر اول: `grandfather`
   - همسر دوم: `grandmother`

4. **افزودن فرزند**
   - شناسه: `father`
   - نام: `علی`
   - جنسیت: `مرد`

5. **افزودن رابطه والد-فرزند**
   - والد: `grandfather`
   - فرزند: `father`
   
6. **افزودن رابطه والد-فرزند دوم**
   - والد: `grandmother`
   - فرزند: `father`

حالا می‌توانید نسبت بین `father` و `grandfather` را جستجو کنید و نتیجه "پدر" را ببینید!

## نکات مهم

- ✅ همیشه برای هر فرد یک شناسه یکتا استفاده کنید
- ✅ برای فرزند، باید هر دو والد (پدر و مادر) رو اضافه کنید
- ✅ اول افراد را اضافه کنید، سپس روابط را
- ❌ نمی‌توانید یک فرد را به عنوان والد خودش اضافه کنید
- ❌ نمی‌توانید یک فرد را به عنوان همسر خودش اضافه کنید

## امکانات کلیدی

### نسبت‌های قابل شناسایی

- **مستقیم**: پدر، مادر، پسر، دختر، همسر

- **دو واسطه**: عمو، عمه، دایی، خاله، برادر/خواهر شوهر/زن
- **سه واسطه**: پسر/دختر عمو، پسر/دختر دایی، پسر/دختر خاله، پسر/دختر عمه
